A tool for engaging teachers in interactive discourse
for learning and practicing goal-directed behaviors and techniques.


Enter session



This demo ONLY runs in the Chrome Desktop Browser

You can ask:

• Chris, what does 2 + 2 equal?
• Luis, please wave
• Everyone, please go to sleep
• Jazmin, please wakeup


If you refer to a student by name, they will answer directly.
You need to click the green START button to start session and press and hold spacebar to talk to class.

There is more information in the help (?) button at the bottom of the screen.
